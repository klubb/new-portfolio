SCUFF

Direct Message App

Description

My first full stack project, Scuff is a direct message app where users can send text, images, and emojis in real time. Users also have the ability to add contacts for quick messaging, change between a light and dark theme, and toggle expletive word filtering.

Technologies

React, Redux, HTML, Styled Components, Socket.io, Node, Express, Postgresql, Auth0

VR ART GALLERY

Virtual Reality App

Description

A virtual reality application where users can browse galleries made by other users or create their own. When creating a gallery, users can upload up to 15 images for each gallery and choose from a selection of music and textures to accompany each gallery.

NOTE: If the preview is not loading you can find a fix under the "Performance Issues" section on the github readme.

Technologies
